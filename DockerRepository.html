Docker Repository

Step 1

Install Dock on your computer, you will need it to complete this task.
Download Docker At:
https://docs.docker.com/get-docker/

Step 2

Pull an image from your computer, and use Apache to create a container for it to be placed into. Consider entering the command: run “docker pull ubuntu”, and to check to see if the image exists in your computer, the command, “docker images”.

Step 3

Run the image you’re trying to work with, this will start the container, and then log into it. Consider using the command: “docker run -it ubuntu:latest /bin/bash”

Step 4

TTo share data between a Docker container and the host,
Use “docker run -it -p 127.0.0.1:80:80 -v :\Docker tutorial\local_dir:/var/www/html ubuntu-my_apache2 /bin /bash
Be careful that nothing else is running in the ‘80’ port.

Step 5

To access the vi Editor in UNIX, first input the following

root@...:/# pwd

root@...:/# cd /var/www/html/

root@...:/ /var/www/html#

Then create an index.html file, press ‘i’ to edit, you can now edit and type html into the given source code. After you are finished, press “ESC” key to leave

Start apache using “​/etc/init.d/apache2 restart​”
